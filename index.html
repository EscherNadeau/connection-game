The rules

How to play.

1

The mission

You get two corners of cinema — a Start and a Goal. Build a chain of connections between them.

2

The one rule

A person connects to a movie or show they acted in. That's it. Titles never link to titles, people never to people.

3

Weave the web

Search anything. If it connects to something on your board, it sticks. Branch freely — dead ends cost nothing.

4

Connect to win

The moment any path links Start to Goal, you've done it. Fewer links, sweeter win — only the shortest path counts.
